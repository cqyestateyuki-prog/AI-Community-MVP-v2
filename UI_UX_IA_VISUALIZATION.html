关键用户流程 / Key User Flows

1
发现内容 / Content Discovery
用户进入首页，浏览动态流和趋势内容，发现感兴趣的帖子
2
阅读与评估 / Read & Evaluate
点击帖子查看详情，阅读内容，评估质量和实用性
3
互动参与 / Interactive Engagement
对内容进行投票、收藏、评论，参与社区讨论
4
内容创作 / Content Creation
创建和发布自己的提示词内容，分享给社区
5
持续参与 / Continuous Engagement
定期访问，管理个人内容，持续参与社区活动
